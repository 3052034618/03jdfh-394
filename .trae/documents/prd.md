## 1. 产品概述

面向游戏编剧课程学生的纯前端对白树练习器，主打低门槛训练心理恐怖对白设计。学生基于老师给出的限制条件，按三幕式结构撰写对白，并通过回放页以手机聊天或游戏字幕方式体验，同时收集同学"哪里开始害怕"的标记反馈。

- 核心目的：让编剧学生在受控范围内练习"不靠怪物、仅凭对话制造不安"的能力
- 目标用户：游戏编剧课程在读学生（编辑者）及同学（体验者/反馈者），教师（出题者）
- 核心价值：降低对白树设计入门门槛，提供即时智能提示与同伴互评反馈

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 教师（出题者） | 在题目页设置限制条件 | 创建/编辑题目、查看学生作品 |
| 学生（编辑者） | 在编辑页撰写三幕对白 | 编辑对白树、查看智能提示、生成分享链接 |
| 同学（体验者） | 通过分享链接进入回放页 | 体验对白、标记害怕位置 |

### 2.2 功能模块

1. **题目页**：场景描述、限制条件输入、示例题目库
2. **编辑页**：三幕式对白编辑器、智能提示面板、阶段导航
3. **回放页**：手机聊天/游戏字幕双模式播放、害怕标记、反馈统计

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 题目页 | 场景设定 | 填写场景标题、情境描述（如"主角独自接到母亲来电，但母亲已去世"） |
| 题目页 | 限制条件 | 添加禁止项（如"不能出现怪物"）和约束（如"只能靠对话制造不安"） |
| 题目页 | 示例题库 | 预置 3-5 道经典心理恐怖题目供快速选用 |
| 编辑页 | 阶段导航 | 开场寒暄 / 异常暗示 / 认知崩塌三阶段切换 |
| 编辑页 | 对白编辑区 | 编写 NPC 台词与玩家选项（每阶段最多 3 个玩家回答） |
| 编辑页 | 智能提示面板 | 实时检测重复恐吓词、NPC 过度解释、缺少情绪后果 |
| 编辑页 | 预览缩略 | 侧栏显示当前阶段对白树结构缩略图 |
| 回放页 | 播放模式切换 | 手机聊天气泡样式 / 游戏字幕打字机样式 |
| 回放页 | 对白播放 | 逐条播放对白，玩家点击选项推进分支 |
| 回放页 | 害怕标记 | 双击/长按某条对白标记"这里开始害怕" |
| 回放页 | 反馈统计 | 展示所有体验者的害怕标记分布热力图 |

## 3. 核心流程

1. 教师（或学生自行）在题目页创建练习题目，设定场景与限制条件
2. 学生进入编辑页，按三幕式结构依次填写开场寒暄、异常暗示、认知崩塌三个阶段的 NPC 台词与玩家选项
3. 编辑器实时给出智能提示（重复恐吓词、NPC 过度解释、缺少情绪后果）
4. 学生完成编辑后生成分享链接
5. 同学通过链接进入回放页，选择播放模式体验对白
6. 体验者在感到害怕的对白处标记
7. 编辑者查看反馈统计，了解"害怕曲线"分布

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：暗色心理恐怖氛围，整体偏黑灰冷色调，点缀暗红/锈色作为警示色
- **主色**：深炭灰 `#1a1a2e`，暗夜蓝 `#16213e`
- **辅助色**：锈红 `#c84b31`（警示/害怕标记），苍白 `#e8e8e8`（正文）
- **按钮风格**：微圆角、扁平化、hover 时轻微发光效果
- **字体**：标题使用有手写/不安感的展示字体（如 Shadows Into Light 或 Creepster），正文使用清晰等宽感字体（如 JetBrains Mono 或 Source Code Pro）
- **布局**：居中窄栏为主，两侧留黑，模拟剧本纸面感
- **图标**：使用 lucide-react 图标库，偏线稿风格
- **氛围细节**：背景添加微弱噪点纹理、对白出现时带轻微打字机动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 要素 |
|----------|----------|---------|
| 题目页 | 场景设定 | 居中卡片式表单，深色背景，输入框带微弱红色边框聚焦效果 |
| 题目页 | 限制条件 | 标签式输入，每条限制以暗红标签呈现，可删除 |
| 题目页 | 示例题库 | 底部横向滚动卡片，hover 时卡片边缘微弱发光 |
| 编辑页 | 阶段导航 | 顶部三步进度条，当前阶段高亮为锈红，已完成阶段带勾号 |
| 编辑页 | 对白编辑区 | 中央主编辑区，NPC 台词居左灰底气泡，玩家选项居右暗红边框卡片 |
| 编辑页 | 智能提示面板 | 右侧固定面板，提示项以图标+文字卡片展示，问题项标红色，良好项标绿色 |
| 编辑页 | 预览缩略 | 左侧可折叠侧栏，迷你树状图 |
| 回放页 | 播放模式切换 | 顶部切换按钮组，手机图标/字幕图标 |
| 回放页 | 对白播放 | 手机模式：全屏模拟聊天界面；字幕模式：黑底居中逐字打出 |
| 回放页 | 害怕标记 | 双击对白后出现暗红脉冲动画 + 骷髅小图标 |
| 回放页 | 反馈统计 | 对白列表右侧显示竖向热力条，标记多的对白颜色更浓 |

### 4.3 响应式设计

- 桌面优先设计，确保在 1280px+ 宽度下体验最佳
- 回放页手机聊天模式在移动端自适应为全屏聊天体验
- 编辑页在窄屏下智能提示面板折叠为底部抽屉

### 4.4 动效设计

- 页面切换：淡入淡出 + 微弱抖动（模拟不安感）
- 对白出现：打字机逐字显示效果
- 害怕标记：暗红脉冲扩散动画
- 阶段切换：滑动过渡
- 智能提示出现：从右侧滑入，带轻微震动
